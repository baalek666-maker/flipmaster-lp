# Module 1 — Trouver les bonnes cartes → Critères 1 & 2 du Framework

> Objectif : Tu sais exactement quoi chercher, où chercher, et comment filtrer les opportunités en moins de 15 minutes par session.
> 🔗 Framework : Ce module construit les **Critères 1 et 2** de la Checklist de Risque — les deux premiers filtres que tu passes chaque carte/lot devant.

---

## 🎯 Critère 1 — La demande est-elle réelle ?

Le premier filtre du framework. La question la plus importante : **est-ce que quelqu'un veut vraiment acheter cette carte ?**

Une carte peut être rare, belle, dans un set populaire — si personne ne l'achète, elle ne vaut rien pour toi. Le Critère 1 élimine les cartes qui semblent intéressantes mais qui stagnent.

**Comment évaluer le Critère 1 :**
- 🟢 **VERT** : Ventes complétées régulières (5+ par mois) sur au moins 2 plateformes
- 🟡 **PRUDENCE** : Ventes complétées mais faibles (1-4 par mois) ou sur 1 seule plateforme
- 🔴 **ROUGE** : Aucune vente complétée dans les 30 derniers jours

---

### Leçon 1.1 — Comprendre le marché Pokémon

Avant de chercher des deals, tu dois comprendre ce qui fait la valeur d'une carte. Ce n'est pas la rareté. C'est la **demande**.

#### L'écosystème des cartes Pokémon

Une carte Pokémon est définie par :
- **Le set** : l'extension dans laquelle elle est sortie (ex : Obsidian Flames, Paldea Evolved, 151)
- **La rareté** : Common, Uncommon, Rare, Holo Rare, Ultra Rare, Secret Rare...
- **Le type** : carte classique, holo, reverse holo, V, VMAX, VSTAR, ex, illustration rare, etc.

#### Pourquoi certaines cartes valent plus que d'autres

La valeur ne dépend pas de la rareté. Elle dépend de la **demande**.

| Facteur | Impact sur la valeur | Exemple |
|---------|---------------------|---------|
| Jouabilité en tournoi | ⬆️⬆️⬆️ | Une carte VMAX jouable en compétition vaut plus qu'une illustration rare qui ne se joue pas |
| Popularité du Pokémon | ⬆️⬆️ | Charizard, Umbreon, Pikachu valent toujours plus que les mêmes raretés d'autres Pokémon |
| Beauté de l'illustration | ⬆️⬆️ | Les "alt arts" et illustration rares sont très demandées par les collectionneurs |
| Rareté de l'impression | ⬆️ | Les pull rates faibles augmentent la valeur... mais seulement s'il y a de la demande |
| Set popularité | ⬆️ | Les sets très attendus (151, Crown Zenith) ont plus de demande que les sets "remplissage" |

**La leçon :** Une carte Ultra Rare d'un Pokémon impopulaire dans un set qui n'intéresse personne vaut moins qu'une Rare Holo de Charizard. La demande > tout.

#### Les sets rentables en ce moment

Le marché bouge. Un set populaire aujourd'hui peut stagner dans 3 mois. La règle : vérifie les ventes complétées, pas les annonces.

Cependant, voici les catégories de sets qui ont historiquement la meilleure demande :

| Catégorie de set | Demande | Pourquoi |
|-----------------|---------|----------|
| Sets anniversaires / spéciaux | ⬆️⬆️⬆️ | Forte nostalgie, collectionneurs nombreux (ex : 151, Crown Zenith) |
| Sets de la saison en cours | ⬆️⬆️ | Joueurs actifs, demande de tournoi |
| Sets vintage (Base Set → Neo) | ⬆️⬆️⬆️ | Collectionneurs, valeur nostalgia, rareté croissante |
| Sets "remplissage" | ⬆️ | Peu de cartes recherchées, beaucoup de bulk |

**Comment rester à jour :** Les sets rentables changent. C'est pour ça que la Checklist commence par le Critère 1 — tu vérifies la demande en temps réel, pas sur une liste figée.

#### Les sets à éviter

- Sets surproduits : trop de booster ouverts = trop de cartes en circulation = prix bas
- Sets avec peu de cartes recherchées : 1-2 bonnes cartes noyant dans du bulk
- Sets en fin de vie : les prix baissent quand le set suivant sort

**Le test simple :** Regarde les ventes complétées des 5 cartes les plus chères du set. Si elles se vendent mal, passe ton chemin.

### Leçon 1.2 — Les signaux de demande

Comment vérifier la demande en 30 secondes. C'est le cœur du Critère 1.

#### La méthode en 3 étapes

**Étape 1 : Vérifie les ventes complétées sur eBay**

1. Va sur ebay.fr
2. Recherche le nom exact de la carte
3. Filtre par "Ventes réalisées" (Sold listings) dans les filtres avancés
4. Regarde les 30 derniers jours

Ce que tu cherches :
- **Nombre de ventes** : 5+ par mois = 🟢, 1-4 = 🟡, 0 = 🔴
- **Prix des ventes** : stable ou en hausse = bon signe. En baisse = prudence.
- **Vitesse de vente** : les listings disparaissent vite = forte demande

**Étape 2 : Vérifie le volume sur Cardmarket**

1. Va sur cardmarket.com
2. Recherche la carte
3. Regarde le nombre de copies disponibles ET le nombre de ventes récentes

Ce que tu cherches :
- **Peu de copies disponibles + ventes régulières** = 🟢 (demande > offre)
- **Beaucoup de copies + ventes lentes** = 🟡 (offre > demande)
- **Beaucoup de copies + aucune vente** = 🔴 (personne ne veut)

**Étape 3 : Vérifie la tendance sur TCGPlayer**

1. Va sur tcgplayer.com
2. Recherche la carte
3. Regarde le "Market Price" et l'historique

Ce que tu cherches :
- **Prix stable ou montant** = 🟢
- **Prix en baisse légère** = 🟡
- **Prix en chute libre** = 🔴

#### Le piège du prix listing vs prix de vente réel

C'est l'erreur #1 des débutants. Le prix auquel une carte est **listée** n'est pas le prix auquel elle se **vend**.

Exemple :
- Une carte est listée à 50€ sur eBay
- Les ventes complétées montrent qu'elle se vend à 35€ en moyenne
- Ton prix de revente réaliste est 35€, pas 50€

**Toujours utiliser les ventes complétées comme référence.** Jamais les prix listing.

#### Les cartes qui se vendent VITE vs les cartes qui stagnent

| Se vend VITE 🟢 | Stagne 🔴 |
|-----------------|----------|
| Cartes V/VMAX/VSTAR jouables en tournoi | Cartes holo de Pokémon impopulaires |
| Alt arts / illustration rares de Pokémon populaires | Cartes de sets surproduits |
| Cartes vintage PSA 9-10 | Bulk commun et peu commun |
| Cartes de set populaire en fin de production | Cartes de set récent encore en production |

#### La règle du Critère 1

**Si pas de ventes complétées dans les 30 derniers jours → 🔴 Critère 1.**

Pas de débat. Pas de "mais elle est belle". Pas de "elle va sûrement monter". Si personne n'achète, tu n'achètes pas. Point.

### Leçon 1.3 — Les keywords de recherche

Trouver les bons deals, c'est 80% la recherche. Si tu cherches comme tout le monde, tu trouves comme tout le monde — c'est-à-dire rien. Les keywords sont ton avantage compétitif.

#### La méthode de dérivation

Ne tape jamais juste "carte Pokémon" ou "Pokémon card". C'est ce que font les 95% des acheteurs. Ils voient les mêmes listings et se battent sur les mêmes deals.

La méthode de dérivation part du **résultat souhaité** et remonte vers le **listing du vendeur** :

1. Tu sais quelle carte tu cherches (le Module 1.1 t'a appris à identifier les cartes demandées)
2. Tu identifies comment un vendeur ignorant pourrait la lister
3. Tu cherches avec SES mots, pas les tiens

Exemple :
- Tu cherches : Charizard VSTAR #174 Brilliant Stars
- Le vendeur ignorant liste : "carte pokemon brillante étoile" ou "pokemon charizard etoile"
- Ton keyword : `"pokemon etoile" charizard` ou `"brillante etoile" V`

#### Keywords eBay — les formules gagnantes

**Pour les lots :**
- `"lot pokemon" VMAX FR`
- `"lot pokemon" holos -booster -pack`
- `"bulk pokemon" rare -common`
- `"collection pokemon" -booster -display`
- `"pokemon cards" lot VF -reprint`

**Pour les cartes individuelles sous-évaluées :**
- Nom du Pokémon + numéro du set (ex : `umbreon 215`)
- Nom du set en FR mal orthographié (ex : `"brillant stars"` au lieu de `"brilliant stars"`)
- `"pokemon" + nom de la carte + `VF` ou `FR`

**Filtres à ajouter systématiquement :**
- Prix max : ton budget par achat (règle des 30%)
- Format : Enchères + Achat immédiat
- Localisation : France + UE (évite les frais de douane)
- État : Neuf + Occasion (sauf si tu cherches du grading)

#### Keywords Vinted / LeBonCoin

Vinted et LeBonCoin sont des mines d'or parce que les vendeurs ne connaissent pas la valeur de leurs cartes.

**Keywords Vinted :**
- `"carte pokemon"` + le nom du Pokémon
- `"lot pokemon"` + rareté
- `"collection pokemon"` (les vendeurs qui vident une collection ne listent pas carte par carte)
- `"cartes pokemon a donner"` (parfois des lots sous-évalués)

**Keywords LeBonCoin :**
- `"jeu de cartes pokemon"` (les vendeurs utilisent souvent cette catégorie)
- `"collection cartes pokemon"` + ville (pour les deals locaux sans shipping)

#### Les mots à ÉVITER dans tes recherches

Ces mots attirent les autres revendeurs — la concurrence monte, les prix montent :

- ❌ `"investissement pokemon"`
- ❌ `"carte pokemon rare valeur"`
- ❌ `"pokemon graded"` ou `"PSA"`
- ❌ `"pokemon first edition"`
- ❌ Tout mot qui montre que le vendeur connaît la valeur

Tu veux les listings où le vendeur **ne sait pas** ce qu'il vend. C'est là que sont les deals.

#### Les recherches sauvegardées

Une fois que tu as tes keywords gagnants, sauvegarde-les :

**Sur eBay :** Mon eBay → Recherches sauvegardées → Créer une recherche → Activer les alertes email

**Sur Vinted :** Sauvegarde tes recherches → Active les notifications "Nouveau listing"

**Le réflexe :** Chaque matin et chaque soir, check tes alertes. Les meilleurs deals partent en moins d'une heure.

## 🎯 Critère 2 — Le prix d'achat est-il sous-évalué ?

Le deuxième filtre. Le premier vérifie qu'il y a de la demande. Le deuxième vérifie que le prix est en dessous du marché — sinon, pas de marge.

**Comment évaluer le Critère 2 :**
- 🟢 **VERT** : Prix d'achat ≤ 60% du prix de revente estimé (marge confortable)
- 🟡 **PRUDENCE** : Prix d'achat entre 60% et 70% du prix de revente (marge fine)
- 🔴 **ROUGE** : Prix d'achat ≥ 70% du prix de revente (pas de marge après frais)

---

### Leçon 1.4 — Comment évaluer un prix de marché

Le Critère 2 repose sur une comparaison : le prix d'achat vs le prix de marché. Mais comment déterminer le "prix de marché" ? Ce n'est pas ce que tu penses.

#### Le prix de marché n'existe pas

Il n'y a pas de prix officiel d'une carte Pokémon. Il y a :
- Le prix auquel les vendeurs **demandent** (listing price)
- Le prix auquel les acheteurs **paient** (sold price)
- Le prix affiché par les price checkers (estimation algorithmique)

Le seul prix qui compte pour toi, c'est le **prix de vente réel** — ce que les acheteurs ont réellement payé.

#### La méthode de référence en 3 sources

Pour établir un prix de marché fiable, croise 3 sources :

**Source 1 : Ventes complétées eBay (50% du poids)**

C'est ta source principale. Les ventes complétées sont la réalité du marché.

1. Recherche la carte sur ebay.fr
2. Filtre par "Ventes réalisées"
3. Note les 10 dernières ventes
4. Ignore les 2 plus hautes (anomalies) et les 2 plus basses (état médiocre ou vente urgente)
5. Calcule la moyenne des 6 ventes restantes

C'est ton **prix de référence eBay**.

**Source 2 : Cardmarket — prix trend (30% du poids)**

Cardmarket affiche un "Trend Price" qui est la moyenne des ventes récentes sur la plateforme.

1. Recherche la carte sur cardmarket.com
2. Note le "Trend Price"
3. Note aussi le "From" price (le prix de la carte la moins chère en état NM)

Le Trend Price te donne une indication européenne. Le "From" price te donne le plancher.

**Source 3 : TCGPlayer — Market Price (20% du poids)**

Le Market Price de TCGPlayer est une estimation algorithmique basée sur les ventes US.

1. Recherche la carte sur tcgplayer.com
2. Note le "Market Price"
3. Ajuste à la baisse de 10-15% pour le marché FR (le marché US est généralement plus cher)

#### Le calcul du prix de marché

```
Prix de marché = (Prix référence eBay × 0,50) + (Trend Cardmarket × 0,30) + (Market Price TCGPlayer ajusté × 0,20)
```

**Exemple concret :** Umbreon VMAX Alt Art #215 Evolving Skies

- Ventes complétées eBay (moyenne 6 ventes) : 42€
- Trend Cardmarket : 38€
- Market Price TCGPlayer : 55€ → ajusté FR -15% = 46,75€

Prix de marché = (42 × 0,50) + (38 × 0,30) + (46,75 × 0,20) = 21 + 11,40 + 9,35 = **41,75€**

#### Les ajustements par état

L'état de la carte change tout. Un prix de marché est toujours pour une carte en état NM (Near Mint).

| État | Ajustement | Description |
|------|-----------|-------------|
| NM (Near Mint) | 100% | Parfaite, pas de rayures visibles |
| LP (Lightly Played) | -20 à -30% | Légères rayures, bords légèrement usés |
| MP (Moderately Played) | -40 à -50% | Rayures visibles, bords usés |
| HP (Heavily Played) | -60 à -70% | Forte usure, plis possibles |
| DMG (Damaged) | -80%+ | Carte endommagée, pliure nette |

**Si tu achètes une carte LP, ton prix de marché est -20 à -30%.** C'est une erreur courante : acheter une carte LP au prix NM, puis ne pas pouvoir la revendre au prix NM.

#### Quand le prix de marché est imprécis

Parfois, les 3 sources donnent des résultats très différents. Dans ce cas :

- **Écart < 15% entre les sources** → confiance élevée, utilise la moyenne pondérée
- **Écart 15-30%** → confiance moyenne, utilise la source la plus basse comme référence prudente
- **Écart > 30%** → 🟡 PRUDENCE au Critère 2. Le marché est instable sur cette carte.

### Leçon 1.5 — Les signaux d'achat : quand le prix est sous-évalué

Le Critère 2 ne se limite pas à comparer deux prix. Il y a des signaux qui te disent qu'un vendeur a sous-évalué sa carte — et que c'est une opportunité.

#### Les 5 signaux de sous-évaluation

**Signal 1 : Le prix est ≥ 30% en dessous du prix de marché**

C'est le signal le plus évident. Si le prix de marché est 40€ et le vendeur demande 25€, c'est 37,5% en dessous. 🟢 Critère 2.

**Règle : Le seuil de ≥ 30% en dessous du marché = 🟢. Entre 20% et 30% = 🟡. Moins de 20% = 🔴.**

Pourquoi 30% ? Parce que les frais de vente (eBay ~13%, shipping ~3-5€, emballage ~1€) mangent environ 20-25% de ton prix de vente. Si tu n'as pas au moins 30% de marge brute, ton profit net est trop faible.

**Signal 2 : Le vendeur ne connaît pas la valeur**

Indices :
- La description ne mentionne ni le set, ni le numéro de carte
- Les photos sont floues ou mal cadrées
- Le titre est générique ("carte pokemon" sans plus de détails)
- Le lot est vendu "à débarrasser" ou "vide collection"
- Le vendeur a 0 ou très peu d'évaluations

C'est le vendeur idéal. Il ne sait pas ce qu'il vend. 🟢

**Signal 3 : Le lot contient des cartes cachées**

Quand un vendeur liste un "lot pokemon" avec des photos où tu repères des cartes de valeur qu'il n'a pas identifiées.

Comment repérer les cartes cachées :
- Zoome sur les photos du lot
- Cherche les symboles de rareté (étoile, holographique)
- Identifie les cartes V/VMAX/VSTAR par leur bordure noire et l'icône en haut
- Les alt arts ont un style distinctif (illustration pleine page)

**Signal 4 : L'enchère se termine à un créneau défavorable**

Les enchères eBay qui se terminent :
- Tôt le matin (6h-8h) → moins de monde connecté = prix plus bas
- Le dimanche matin → bon créneau
- En semaine pendant les heures de bureau → moins de concurrents

**Signal 5 : Le prix de réserve n'est pas atteint**

Si une enchère a un prix de réserve non atteint et que le prix actuel est bas, tu peux contacter le vendeur après la fin de l'enchère pour négocier directement. Certains vendeurs acceptent de vendre hors plateforme au prix de la dernière enchère.

#### La règle du Critère 2

**Prix d'achat ≤ 60% du prix de revente estimé → 🟢**
**Prix d'achat entre 60% et 70% → 🟡**
**Prix d'achat ≥ 70% → 🔴**

Le seuil de 70% correspond à la zone où les frais de vente mangent tout le profit. En dessous de 60%, tu as une marge confortable même si tu dois baisser ton prix de vente de 10%.

### Leçon 1.6 — Études de cas : Critères 1 & 2 en action

On applique les Critères 1 et 2 de la Checklist sur des situations réelles.

#### Cas 1 : Le lot Vinted sous-évalué 🟢🟢

**Situation :** Tu vois un lot sur Vinted à 30€. La description dit "cartes pokemon vieilles collection". Les photos montrent environ 50 cartes, dont au moins 3 cartes V et 2 reverse holos de sets populaires.

**Critère 1 — La demande est-elle réelle ?**
- Vérification eBay : les 3 cartes V se vendent à 8-12€ chacune en moyenne, ventes complétées régulières
- Cardmarket : Trend Price confirme les prix
- Verdict : 🟢 VENT — les cartes V ont une demande régulière

**Critère 2 — Le prix est-il sous-évalué ?**
- Prix de marché estimé des 3 cartes V seules : 8 + 10 + 12 = 30€
- Les reverse holos et le reste du lot ajoutent ~10-15€ de valeur potentielle
- Valeur totale estimée : ~40-45€
- Prix d'achat : 30€ → 30/45 = 67% du prix de marché
- Verdict : 🟡 PRUDENCE — la marge est fine, mais les cartes cachées dans le lot peuvent améliorer le deal

**Décision :** 1🟢 + 1🟡 = pas encore de décision claire. Il faut les Critères 3-5 (Module 2-3) pour trancher. Mais c'est un deal à investiguer.

#### Cas 2 : La carte "belle mais morte" 🔴🟢

**Situation :** Une carte Ultra Rare d'un Pokémon impopulaire dans un set récent, listée à 15€. Le prix de marché est de 25€.

**Critère 1 — La demande est-elle réelle ?**
- Vérification eBay : 0 vente complétée dans les 30 derniers jours
- Cardmarket : 47 copies disponibles, 2 ventes ce mois
- Verdict : 🔴 ROUGE — personne ne veut acheter cette carte

**Critère 2 — Le prix est-il sous-évalué ?**
- Prix de marché : 25€, prix demandé : 15€ → 60% du marché
- Verdict : 🟢 VERT — le prix est en dessous du marché

**Décision :** 1🔴 + 1🟢 = 1🟢 seulement. Même si le prix est bon, si personne ne veut acheter, tu vas posséder une carte que tu ne peux pas revendre. **PASSER.**

#### Cas 3 : Le Charizard VSTAR sous-évalué 🟢🟢

**Situation :** Charizard VSTAR #174 Brilliant Stars listé à 18€ sur eBay (Buy It Now). Prix de marché : 35€.

**Critère 1 — La demande est-elle réelle ?**
- Vérification eBay : 12 ventes complétées ce mois
- Cardmarket : Trend Price 33€, 8 ventes ce jour
- TCGPlayer : Market Price $42 (ajusté FR ~38€)
- Verdict : 🟢 VENT — demande très forte, Charizard est un Pokémon populaire

**Critère 2 — Le prix est-il sous-évalué ?**
- Prix de marché (calculé) : (35×0,50) + (33×0,30) + (38×0,20) = 17,50 + 9,90 + 7,60 = 35€
- Prix d'achat : 18€ → 18/35 = 51% du prix de marché
- Verdict : 🟢 VERT — 49% en dessous du marché, excellente marge

**Décision :** 2🟢 — le deal passe les Critères 1 et 2. On continue l'évaluation avec les Critères 3-5 dans les modules suivants.
