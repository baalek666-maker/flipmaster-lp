## Module 1 — Trouver les bonnes cartes → Critères 1 & 2 du Framework

> Objectif : L'étudiant sait exactement quoi chercher, où chercher, et comment filtrer les opportunités en moins de 15 minutes par session.
> 🔗 Framework : Ce module construit les **Critères 1 et 2** de la Checklist de Risque — les deux premiers filtres que tu passes chaque carte/lots devant.

### 🎯 Critère 1 — La demande est-elle réelle ?

> Le premier filtre du framework : est-ce que quelqu'un veut vraiment acheter cette carte ?

#### Leçon 1.1 — Comprendre le marché Pokémon
- L'écosystème : sets, raretés, types de cartes (holo, reverse, V, VMAX, VSTAR, ex, etc.)
- Pourquoi certaines cartes valent plus que d'autres (demande > rareté)
- Les sets rentables en ce moment (mise à jour mensuelle dans la communauté)
- Les sets à éviter (surproduits, hype artificiel)
- **Format : Texte + tableau des sets par rentabilité**

#### Leçon 1.2 — Les signaux de demande
- Comment vérifier la demande en 30 secondes : ventes complétées eBay, trending TCGPlayer, volume Cardmarket
- Le piège du prix listing vs prix de vente réel
- Les cartes qui se vendent VITE vs les cartes qui stagnent
- La règle : si pas de ventes complétées dans les 30 derniers jours → 🔴 Critère 1
- **Format : Texte + captures annotées de ventes complétées**

#### Leçon 1.3 — Les keywords de recherche
- Keywords eBay : les formules gagnantes ("lot pokemon VMAX", "bulk holos FR", etc.)
- Keywords Vinted / LeBonCoin
- Comment créer ses propres keywords (méthode de dérivation)
- Les mots à ÉVITER dans tes recherches (qui attirent les autres revendeurs)
- **Livrable : Liste de 50+ keywords testés et mis à jour**
- **Format : Texte + table de keywords classés par canal**

### 🎯 Critère 2 — Le prix d'achat est-il sous-évalué ?

> Le deuxième filtre : le vendeur connaît-il la valeur de ce qu'il vend ?

#### Leçon 1.4 — Les filtres qui changent tout
- Filtres eBay : prix, format (enchère vs achat immédiat), localisation, date de fin
- Le filtre "fin dans 15 min" — pourquoi c'est ton meilleur ami
- Les erreurs de listing des autres : catégories mal classées, typos, photos floues
- Comment repérer un listing sous-évalué en 30 secondes
- **Format : Texte + captures annotées de filtres eBay**

#### Leçon 1.5 — Les signaux d'achat
- Le spread potentiel : comment estimer la valeur d'un lot en 2 minutes
- Les signaux que le vendeur ne connaît pas la valeur (photos mal cadrées, description vague)
- Le timing : quand les meilleurs deals apparaissent (jours, heures)
- Les lots vs les cartes individuelles : quand privilégier quoi
- La règle : si le prix listing est ≥ 70% de la valeur de revente → 🟡 Critère 2 (marge trop fine)
- **Format : Texte + grille de signaux d'achat (tableau)**

#### Leçon 1.6 — Où acheter : les canaux
- eBay (enchères + Buy It Now) — avantages et pièges
- Vinted / LeBonCoin — le marché FR sous-évalué
- Facebook Groups — les deals privés
- Bourses / conventions — le terrain de jeu local
- Cardmarket (UE) — quand l'utiliser
- **Format : Texte + tableau comparatif des canaux (frais, audience, risques)**

**Livrable Module 1 :**
- Liste de 50+ keywords (document + sheet)

## Module 2 — Calculer le spread réel → Critères 3 & 4 du Framework

> Objectif : L'étudiant ne JAMAIS achète sans avoir calculé le profit réel au centime près. Plus de mauvaises surprises.
> 🔗 Framework : Ce module construit les **Critères 3 et 4** de la Checklist de Risque — les deux filtres financiers qui transforment une intuition en décision chiffrée.

### 🎯 Critère 3 — Le spread réel est-il ≥ 30% ?

> Le troisième filtre : après TOUS les frais déduits, est-ce que ça vaut le coup ?

#### Leçon 2.1 — La formule du spread réel
- Spread = Prix de vente estimé − Prix d'achat − Frais de plateforme − Shipping − Taxes − Coût grading (si applicable)
- Exemple chiffré complet : achat 80€ → vente 120€ = ??? de profit réel
- Le piège des 13% (eBay + PayPal)
- Pourquoi une marge brute de 20% = break-even
- La règle : spread réel < 30% → 🔴 Critère 3 (le risque n'en vaut pas la récompense)
- **Format : Texte + tableau de calcul pas-à-pas**

#### Leçon 2.2 — Les frais cachés que personne mentionne
- Frais eBay : commission finale + PayPal + frais d'insertion
- Frais Vinted : protection acheteur
- Shipping : enveloppe à bulles + top loader + suivi = coûts réels
- TVA : quand la déclarer, comment l'intégrer
- Retours : le coût caché des litiges
- **Format : Texte + tableau des frais par plateforme**

#### Leçon 2.3 — Le calculateur de spread Pokévendre Pro
- Comment utiliser le calculateur (walkthrough complet)
- Scénario 1 : Achat lot eBay, revente individuelle
- Scénario 2 : Achat Vinted, revente eBay
- Scénario 3 : Achat avec potentiel grading
- Quand NE PAS utiliser le calculateur (et faire le calcul de tête)
- **Format : Texte + captures annotées du calculateur + 3 scénarios chiffrés**

### 🎯 Critère 4 — Les seuils d'achat sont-ils respectés ?

> Le quatrième filtre : chaque type de carte a un seuil en dessous duquel tu n'achètes pas.

#### Leçon 2.4 — Les seuils d'achat par catégorie
- Règle d'or : n'achète que si le spread réel ≥ 30%
- Seuil minimum par type de carte (holo, V, VMAX, vintage)
- Quand faire une exception (lot avec potentiel grading caché)
- Le seuil psychologique : pourquoi 20% de marge ne suffit JAMAIS
- Tableau des seuils par catégorie (mis à jour mensuellement)
- La règle : seuil non respecté → 🔴 Critère 4 (la carte n'est pas pour toi)
- **Format : Texte + tableau des seuils par catégorie**

#### Leçon 2.5 — Étude de cas : 5 calculs réels
- Cas 1 : Le lot qui semblait bon mais ne l'était pas
- Cas 2 : Le lot ennuyeux avec un gem caché
- Cas 3 : Le calcul grading — quand envoyer à PSA/BGS/CGC
- Cas 4 : Le lot FR vs US — arbitrage géographique
- Cas 5 : Le deal qui semblait trop beau (et l'était)
- Pour chaque cas : application de la Checklist (Critères 1-4) → décision ACHETER ou PASSER
- **Format : Texte + tableaux de calcul détaillés + Checklist remplie pour chaque cas**

**Livrable Module 2 :**

## Module 4 — Revendre au bon moment → Signaux de sortie (GARDER / REVENDRE)

> Objectif : L'étudiant sait exactement comment lister, quand lister, et comment maximiser chaque vente.
> 🔗 Framework : Le Framework de Décision ne s'arrête pas à l'achat. Ce module ajoute les **Signaux de Sortie** — quand GARDER, quand REVENDRE, et comment exécuter la revente pour maximiser le profit.

### 🎯 Signal de Sortie — GARDER ou REVENDRE ?

> Le framework te dit quand ACHETER. Maintenant il te dit quand VENDRE.

#### Leçon 4.1 — Le signal GARDER : quand ne pas vendre
- La carte monte encore → attendre le pic
- Le set n'a pas encore roté → la rareté va augmenter
- Le grading est en cours → la valeur va multiplier
- La règle : si le spread de revente actuel < spread potentiel dans 3 mois → GARDER
- **Format : Texte + tableau de décision GARDER vs REVENDRE**

#### Leçon 4.2 — Le signal REVENDRE : quand sortir
- Le pic est atteint (indicateurs de retournement)
- Une rotation de format approche → vendre AVANT
- La carte a été reprinted → sortir immédiatement
- Besoin de liquidité pour un meilleur deal → la règle du capital immobilisé
- **Format : Texte + checkliste des signaux de sortie**

### 🔧 Exécuter la revente

#### Leçon 4.3 — Listing optimal : la fiche parfaite
- Le titre SEO : la formule qui fait vendre (Nom carte + Set + Numéro + État + Langue)
- Les photos qui vendent (angles, lumière, fond, mise en scène)
- La description qui rassure (transparence = confiance = prix plus élevé)
- Les erreurs qui tuent une vente (photos floues, titre vague, pas de numéro)
- **Format : Texte + captures annotées de listings bons et mauvais + template copier-coller**

#### Leçon 4.4 — Pricing strategy : le bon prix
- Comment fixer son prix (pas le plus bas, pas le plus haut)
- Le prix psychologique : pourquoi 19,99€ > 20€
- L'enchère vs l'achat immédiat : quand choisir quoi
- Le "Make Offer" : quand l'activer, comment l'utiliser
- Ajuster ses prix : quand baisser, quand tenir
- **Format : Texte + tableau de pricing par type de carte**

#### Leçon 4.5 — Le timing de vente
- Vendre juste après un tournoi / événement médiatique
- Vendre avant une rotation de format
- Les saisons : Noël, rentrée, vacances — quand vendre quoi
- Le "hold strategy" : quand garder une carte 6 mois
- **Format : Texte + calendrier des meilleurs moments de vente**

#### Leçon 4.6 — Où vendre : plateforme par plateforme
- eBay : le standard international (avantages, frais, reach)
- Vinted : le marché FR sans frais de vente (mais audience limitée)
- Cardmarket : le marché EU des cartes TCG
- Facebook Groups : vente directe, zéro frais, mais risques
- Chaque plateforme a son moment : croiser les listings
- **Format : Texte + tableau comparatif des plateformes (frais, audience, délais, risques)**

#### Leçon 4.7 — Shipping & logistique
- Emballer une carte correctement (sleeve + top loader + carton)
- Shipping national vs international : coûts et délais
- Le suivi obligatoire : pourquoi c'est non-négociable
- Gérer les litiges : preuves photo, assurance, signatures
- **Format : Texte + guide photo d'emballage étape par étape**

#### Leçon 4.8 — Grading : quand envoyer, à qui, comment
- PSA vs BGS vs CGC : comparatif coûts, délais, valeurs
- Le seuil de rentabilité du grading (calcul réel)
- Quelles cartes envoyer (et lesquelles NE PAS envoyer)
- Le processus pas-à-pas (soumission, envoi, réception, revente)
- **Format : Texte + tableau comparatif PSA/BGS/CGC + calculateur de rentabilité grading**

**Livrable Module 4 :**
- Template de listing eBay (copier-coller)
- Grille photo checklist (document imprimable)
- Comparatif grading PSA/BGS/CGC (coûts + délais)
- **Signaux de Sortie GARDER / REVENDRE (document imprimable)** — le pendant vente de la Checklist d'achat
