Leçon 0.4 — Les outils de base

4 outils. Pas 15.

Tu as ta Checklist avec 5 critères. Chaque critère a besoin d'un outil pour le vérifier. Pas plus.

Le piège : Passer 3h à configurer des outils au lieu de chercher des deals. Les 4 ci-dessous, tu les installes en 10 minutes et tu passes à la suite.

📊

1. Price checkers — Critères 1 & 2

Trois sites, trois usages. Pas besoin de tous les ouvrir à chaque fois.

PriceCharting → pricecharting.com

Le plus utile pour toi. Historique de prix FR + US, ventes complétées.

→ Quand l'utiliser : Vérifier la demande réelle (Critère 1). Pas de ventes récentes = 🔴.

Cardmarket → cardmarket.com

La référence européenne. Prix de marché EU, volume de vente.

→ Quand l'utiliser : Comparer le prix demandé avec le marché (Critère 2).

TCGPlayer → tcgplayer.com

Le plus complet pour le marché US. Utile pour les cartes rares en anglais.

→ Quand l'utiliser : Cartes anglophones ou quand tu vends à l'international.

⚠️ Le piège #1 du price checking : Le prix affiché ≠ le prix de vente. Regarde toujours les ventes complétées, pas les annonces en cours. Un gars peut demander 50€ — si le dernier vendu est à 20€, la vraie valeur est 20€. Détaillé au Module 1.

⚡

2. Extensions Chrome — accélère l'évaluation

Card Ladder

Affiche le prix directement sur la page eBay. Plus besoin d'ouvrir un autre onglet.

→ Installe : Chrome Web Store → « Card Ladder »

Pokémon TCG Player

Infos sur les sets, raretés, prix. Idéal pour identifier une carte en 5 secondes.

→ Installe : Chrome Web Store → « Pokémon TCG Player »

Installe les deux. Redémarre Chrome. Tu es prêt.

🧮

3. Calculateur de Spread — Critères 3 & 4

L'outil qui transforme une impression en chiffre.

Tu entres : prix d'achat, plateforme de vente, prix de vente estimé.

Il calcule : frais de plateforme, shipping estimé, profit net, ROI en %.

Il te dit : le spread est-il ≥ 30% ? 🟢 ou 🔴 ?

4 étapes :

Accès : Le calculateur de spread est disponible dans ton espace membre Pokévendre Pro. Tu le trouveras dans la section « Outils » de la formation. Un tableur Google Sheets est aussi fourni en backup — fais-en une copie pour l'utiliser.

On l'utilise en détail dans le Module 2.

📸

4. Setup photo — pour le Module 4

Pas besoin de matériel pro. Smartphone + lumière naturelle = suffisant.

Le minimum :

📱 Smartphone → 8MP minimum, n'importe quel téléphone récent
☀️ Lumière → naturelle près d'une fenêtre. Pas de flash (reflets garantis)
📄 Fond → feuille blanche ou tapis de souris noir uni
📚 Support → pile de livres ou trépied improvisé pour la stabilité

Les 4 angles obligatoires :

On détaille les photos de vente dans le Module 4.
