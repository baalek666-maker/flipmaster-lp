### Architecture:
- Auth flow: POST `/api/verify-code` { code } → { status: "ok", token }
- localStorage: `pv_access_code`, `pv_access_token`
- Module content: POST `/api/module/:id` { code, token } → { status: "ok", html }
- Sidebar: desktop fixe gauche (280px), mobile drawer overlay
- Contact email: pokevendrepro@gmail.com
- ModuleInfo interface: `{ id: string; emoji: string; label: string; title: string }`
